# AI Domination

A pixel-art arena game where the robot adapts its strategy based on the player's actions.

## How It Works
- Player can attack, dodge, or wait
- Boss tracks player behavior
- Boss increases counters to repeated actions
- Strategy changes each round

## Tech Stack
- JavaScript + HTML Canvas
- Pixel art sprites
- Rule-based adaptive AI

## Team Roles
- Game Logic & Player Controls: Person 1
- AI Boss & Learning Logic: Person 2
- Pixel Art, Rendering & UX: Person 3

## How to Run
Open index.html using a local server (Live Server recommended).
